## Method-specific DID syntax

The following ABNF defines the blobaa-specific DID scheme:

```ABNF
blobaa-did = "did:baa:" blboaa-identifier
blobaa-identifier = [ ardor-network ":" ] ardor-tx-hash
ardor-network = "m" / "t"
ardor-tx-hash = 64HEXDIG
```
